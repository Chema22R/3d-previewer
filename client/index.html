Objects Manipulation (mouse)

Rotation - left click

Panning - right click

Zoom - scroll wheel

Objects Manipulation (keyboard)

Rotation - arrow keys

Panning - W, A, S, D keys

Zoom - Z, X keys

Objects Manipulation (tactile)

Rotation - one finger movement

Panning - three finger movement

Zoom - two fingers pinching

Keyboard Shortcuts

u - upload files to server (+ enter)

f - file list opening

s - search for specific files

h - help menu deployment

* trigger commands: Alt, Alt+Shift or Control+Alt

Tab Index

1 - upload files to server

2 - file list opening

3 - search for specific files

* needed to press enter to execute
